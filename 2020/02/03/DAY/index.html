加载一张图片插在文章中都不是一件容易的事情

大致总结了一下这方面的坑

我看的是博客上面的

第一步：配置文件_config.yml 里的post_asset_folder:这个选项设置为true（这个文件在主页里面的_config.yml）

第二步：是在git中安装一个上传本地图片的插件：npm install https://github.com/CodeFalling/hexo-asset-image –save（一定是这样的一句话哈，这是一个关键，其他版本的会报错）

第三步：hexo new “随便文件夹”；这个 时候在你的根目录下仔细的去看source目录里面（_posts）下面会多出一个与你新建文件夹同名的文件夹，你把图片放在这个文件夹里面；

第四步：在你写博客文章的地方写上：![]（文件夹名/图片名.图片格式）

第五步：在博客目录中找到public文件夹，看看你的新建的文件的index.html里面的H5写法,找到你图片的位置，核实图片路径是否正确；

OK了，就可以 hexo clean，然后重新 生成了； hexo g hexo s 预览成功；

总结就是了：装，新建，编写，生成，检查，预览，发布:
